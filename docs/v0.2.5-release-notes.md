# WaveYourYarn v0.2.5 发布说明

> 开发完成日期：2026-07-20
>
> 发布状态：首个 GitHub Release；提供 macOS arm64 与 x64 未签名开发包。

## 版本定位

v0.2.5 是发行与质量基础版本。它不增加 AI、远端写入或 Apple Music 功能，而是让现有网易云只读数据闭环具备可打包、可检测、可恢复和可诊断的工程基础。

## 下载选择

- Apple Silicon（M1/M2/M3/M4 等）选择 `mac-arm64`；
- Intel Mac 选择 `mac-x64`；
- DMG 适合常规拖拽安装，ZIP 适合直接解压测试；
- `SHA256SUMS.txt` 用于核对下载文件完整性。

当前产物未进行 Apple Developer ID 签名和公证。首次启动时需要在 macOS“隐私与安全性”中确认打开，仅应从项目官方 Release 页面下载。

## 主要变化

### Electron 运行时与安全

- 修复并固化 sandbox preload 的 CommonJS 构建与产物校验；
- 使用真实 Electron 生产构建验证 preload、IPC ping、主路由、备份和诊断；
- 主窗口和登录窗口限制导航、弹窗、外链和权限请求；
- 所有业务 IPC 统一校验主 frame 与受信 renderer URL；
- renderer 增加 CSP，IPC 错误统一脱敏后返回。

### 备份与恢复

- 新增版本化 `.wyybackup` 格式，包含 manifest、数据数量、大小和 SHA-256；
- 使用 SQLite Online Backup 生成一致性快照；
- 用户凭据和 `secret:` 设置不会进入普通备份；
- 恢复前验证文件头、校验值、schema、完整性和孤立关系；
- 恢复前自动保存当前数据库，文件替换失败时执行回滚；
- 文件路径只由 Electron 原生对话框选择，不暴露给 renderer。

### 诊断能力

- 新增 JSONL 结构化日志、2 MiB 滚动和最多 3 个历史文件；
- Cookie、Authorization、Bearer、Token、API Key、URL query 和用户目录名统一脱敏；
- 设置页显示运行时、数据库、存储和 adapter 摘要；
- 用户可以主动导出脱敏 JSON 诊断包，应用不会自动上传。

### 打包与维护性

- 接入 electron-builder，固定 bundle id、产物命名、asar 和原生模块配置；
- 新增 macOS arm64/x64 原生 runner 标签构建工作流，生成未签名 DMG/ZIP；
- 新增项目图标 SVG、可重复生成的 ICNS 脚本和打包配置；
- `ExportPage` 拆分为设置、预览、历史和排序模块；
- `StatisticsService` 拆出可独立测试的纯统计算法；
- 统一 IPC result 封装，加入 v0.2.4 数据库兼容夹具。

## 数据兼容与升级

- 当前数据库 schema 仍为 6，v0.2.4 数据库无需破坏性迁移；
- 启动时仍由 migration runner 检查并补齐缺失迁移；
- 从更高 schema 版本创建的备份会被拒绝，避免新数据被旧应用覆盖；
- 普通备份不包含登录凭据，恢复后需要重新登录网易云。

## 自动化覆盖

- URL、IPC sender、备份格式、文件回滚、恢复 token、脱敏、诊断、统计和导出排序单元测试；
- 新库、v0.2.3 迁移和代表性 v0.2.4 数据库兼容测试；
- 真实 Electron 源态与打包态冒烟测试；
- E2E 注入假凭据并验证备份及诊断全文不存在原值；
- CI 使用 Node 22，macOS 标签任务分别在 arm64 与 Intel runner 构建。
- arm64 原生包和 x64 Rosetta 包均已执行完整打包态 E2E。

## 已知限制

- 产物尚未进行 Apple Developer ID 签名和公证，macOS 会显示未签名应用提示；
- 自动更新尚未连接发布源；
- Windows 正式打包不在本版本范围；
- 网易云登录可能受上游页面和风控变化影响，仍保留手动 Cookie 兜底；
- 正式发布前必须执行 [v0.2.5 人工回归清单](v0.2.5-manual-regression.md)。

## 回退方式

如升级后遇到阻断问题，先退出应用并复制当前用户数据目录，再保留诊断包。若刚执行过恢复，可使用 `recovery/` 中的恢复前数据库；不要在应用运行时直接替换 SQLite、WAL 或 SHM 文件。

# WaveYourYarn v0.2.5 发布前人工回归清单

> 适用版本：v0.2.5
>
> 清单状态：待在 arm64、x64 和真实网易云账号环境执行
>
> 原则：只使用测试账号或已确认可操作的个人账号；本清单不执行远端写入、删除或覆盖。

## 1. 测试矩阵

| 环境        | 安装包             | 账号         | 结果   |
| ----------- | ------------------ | ------------ | ------ |
| macOS arm64 | unsigned DMG / ZIP | 真实个人账号 | 待执行 |
| macOS x64   | unsigned DMG / ZIP | 真实个人账号 | 待执行 |
| macOS arm64 | v0.2.4 数据库升级  | 可离线       | 待执行 |
| macOS x64   | v0.2.4 数据库升级  | 可离线       | 待执行 |

执行前记录 macOS 版本、芯片架构、安装包文件名和 SHA-256。对现有用户数据库测试前，先复制整个 WaveYourYarn 数据目录。

## 2. 安装与启动

- [ ] DMG 可以挂载，应用可以复制到 Applications；
- [ ] ZIP 解压后的应用可以启动；
- [ ] macOS 对未签名开发包的风险提示与发布说明一致；
- [ ] 首次启动无白屏，侧栏和首页可见；
- [ ] 页面不出现 `Electron preload API is unavailable`；
- [ ] 首页显示 `IPC ping result: pong`；
- [ ] 重启后仍能正常初始化数据库和日志；
- [ ] v0.2.4 数据库启动后歌曲、歌单、关系、设置和导出历史数量不减少。

## 3. 网易云只读闭环

- [ ] 打开网页登录窗口，不加载 WaveYourYarn preload；
- [ ] 网页登录成功后能识别当前用户；
- [ ] 若网页登录受风控限制，手动 Cookie 登录兜底可用；
- [ ] 同步“我喜欢的音乐”，数量、顺序和时间字段合理；
- [ ] 同步创建/收藏的歌单列表；
- [ ] 同步单个歌单和全部歌单歌曲；
- [ ] 搜索、排序、详情和统计页面可用；
- [ ] CSV、JSON、Markdown 导出均能完成并打开文件；
- [ ] 退出登录后同账号缓存仍可离线读取；
- [ ] 整个过程没有创建、删除、覆盖或修改远端歌单。

## 4. 备份与恢复

- [ ] 设置页可以创建 `.wyybackup`；
- [ ] 备份预览显示版本、schema 和核心数量；
- [ ] 备份二进制内容中检索不到 Cookie、Token 或 `secret:` 设置；
- [ ] 损坏备份、非备份文件和高版本 schema 会在覆盖数据库前被拒绝；
- [ ] 恢复必须再次确认，且 renderer 不接触真实文件路径；
- [ ] 恢复前在 `recovery/` 生成当前数据库应急快照；
- [ ] 恢复后应用自动重启；
- [ ] 歌曲、歌单、关系、公开设置和导出历史数量与备份 manifest 一致。

## 5. 诊断与安全

- [ ] 设置页显示应用、Electron、Node、Chromium、架构和 schema；
- [ ] 导出的 JSON 诊断包可以正常解析；
- [ ] 诊断包只包含数量摘要，不包含歌曲名、歌手或专辑明细；
- [ ] 全文检索不到 Cookie、Authorization、Bearer、Token、API Key 和用户目录名；
- [ ] 应用日志滚动后最多保留当前文件及 3 个历史文件；
- [ ] 主窗口导航到非受信 URL 被拒绝；
- [ ] 登录窗口跳转到非网易云受信域被拒绝；
- [ ] 摄像头、麦克风、位置等权限请求默认被拒绝；
- [ ] 外链只会在明确允许的 HTTPS 域名上打开。

## 6. 结果记录

每个失败项至少记录：架构、macOS 版本、步骤、预期、实际结果、脱敏诊断包文件名和是否阻断发布。严禁把 Cookie、Token、完整个人路径或真实音乐明细提交到公开 issue。

该清单用于首发后的真实账号与系统兼容性跟踪；发现阻断问题时，应暂停后续版本推广并在修复后重新执行对应矩阵。

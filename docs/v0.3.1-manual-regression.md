# WaveYourYarn v0.3.1 AI 音乐报告人工回归清单

> 适用分支：`codex/v0.3-llm-foundation`
>
> 适用范围：v0.3.0 LLM 基础与 v0.3.1 AI 音乐报告；v0.3.2 报告导出不在本清单内。
>
> 安全原则：只读取网易云账号数据。报告删除是本地软删除，任何步骤都不得创建、修改或删除远端歌曲与歌单。

## 1. 执行前准备

- [ ] 确认当前分支为 `codex/v0.3-llm-foundation`；
- [ ] 执行 `pnpm install` 与 `pnpm run rebuild:native`；
- [ ] 执行 `pnpm run check`，确认类型检查、lint、测试和生产构建通过；
- [ ] 执行 `pnpm run test:e2e`，确认本地 mock provider 四条报告路径通过；
- [ ] 使用测试账号或已确认可操作的个人账号，先备份本地 WaveYourYarn 数据目录；
- [ ] 准备一个用户自行选择的兼容 API，确认费用、隐私和数据处理条款；不要把真实 Key 写入仓库、截图、日志或 issue。

## 2. 模型配置与连接

- [ ] 通过 `pnpm run dev` 启动应用，侧栏可以进入“设置”；
- [ ] 新建模型配置时可填写名称、Base URL、模型 ID、超时、输出模式、语言与歌曲上限；
- [ ] 保存 API Key 后界面只显示“已保存 Key”，不能重新读取明文；
- [ ] “测试连接”成功时只产生连接测试任务，不上传歌曲数据；
- [ ] 错误 Key 显示安全的鉴权失败信息，不显示响应体、Authorization 或 Key；
- [ ] 修改 Base URL、模型 ID 或歌曲上限后，旧披露预览不能继续用于生成；
- [ ] 删除 Key 后不能生成报告；重新设置 Key 后恢复可用。

## 3. 数据准备与披露确认

- [ ] 登录并只读同步“我喜欢的音乐”；至少准备一首本地缓存歌曲；
- [ ] 如需验证歌单来源，先只读同步一个歌单及其歌曲；
- [ ] 侧栏进入“AI 音乐报告”，可选择 liked、全部缓存或已同步歌单；
- [ ] 分析歌曲数只能填写 1 到当前 profile 上限之间的整数；
- [ ] 报告语言可修改，发送前预览显示最终语言；
- [ ] 预览显示目标 origin、模型、来源、实际歌曲数、上限与字段清单；
- [ ] 未勾选明确确认时不能发送；仅打开预览或创建授权不会上传歌曲；
- [ ] 记住授权后，相同范围可以减少重复确认；模型、目标、来源、语言、字段或数量范围变化后必须重新确认；
- [ ] 设置页撤销已记住授权后，下一次生成必须重新确认。

## 4. 报告生成与任务状态

- [ ] 确认发送后任务依次展示整理事实、等待模型、校验与本地保存进度；
- [ ] 成功后自动打开新报告，历史列表增加一条；
- [ ] 报告显示模型、服务 origin、Prompt 版本、Content schema、dataset digest 与生成时间；
- [ ] 报告明确说明仅基于收藏样本、用于娱乐和自我回顾，不是心理诊断；
- [ ] 口味速写、人格称号、四个偏好维度和各观察项均显示至少一个有效 evidence；
- [ ] evidence 中的歌曲 ID 只来自本次样本，事实字段只来自允许的确定性事实清单；
- [ ] liked 缺少可靠时间时，界面明确说明“最近”使用收藏或歌单顺序近似；
- [ ] 等待模型时点击取消，任务进入 cancelled，历史列表不增加成功报告；
- [ ] 失败或取消后选择重试，必须获得新的披露授权并创建新的 job；
- [ ] 超时、鉴权失败、模型不存在和非法输出只显示安全错误，不保存成功报告。

## 5. 历史、本地管理与离线

- [ ] 重启应用后仍可读取报告历史和详情；
- [ ] 断开网络后仍可打开已经保存的报告；
- [ ] 重命名只改变本地报告标题；
- [ ] “用相同来源重新生成”会回到生成区并重新执行披露流程；
- [ ] 本地源缓存存在时，相同来源可生成新的独立报告；
- [ ] 清理 liked 或歌单缓存不会删除已有报告及其来源快照；
- [ ] 删除报告前出现本地确认，删除后该报告从历史隐藏；
- [ ] 删除报告不会删除歌曲、歌单、其他报告或任何网易云远端内容；
- [ ] 删除模型配置后，既有报告仍可离线阅读，并保留生成时模型与服务审计信息。

## 6. 自动化与发行边界复核

- [ ] `pnpm run verify:dev-llm-boundary` 通过；
- [ ] `pnpm run package:dir` 后执行当前包的开发配置泄露扫描并通过；
- [ ] Release 目录不包含 `.env`、本地开发 Base URL、模型 ID 或 API Key；
- [ ] Electron E2E 只连接本机 mock provider，不访问真实付费模型；
- [ ] 应用不存在网易云远端创建、更新、删除、重命名、添加歌曲或移除歌曲入口。

## 7. 结果记录

记录操作系统、架构、分支提交、模型协议类型、是否本地/公网服务、来源类型、样本数量、任务结果和脱敏诊断包文件名。不得记录真实 API Key、Authorization、Cookie、完整个人路径或可识别的私人音乐明细。

发现以下任一情况时停止发布：未经确认上传音乐数据；renderer 可读取 Key；失败或取消任务产生成功报告；本地报告操作影响网易云远端内容；清理缓存连带删除历史报告；发行产物包含开发模型配置。
